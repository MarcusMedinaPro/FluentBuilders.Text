# Getting Started

Get started with `MarcusMedina.Fluent.Text.Core.Core` in a C# project.

## Installation

```bash
dotnet add package MarcusMedina.Fluent.Text.Core.Core
```

## Requirements

- net8.0 or compatible target
- Nullable reference types enabled when possible

## Basic Usage

Start by referencing the package namespaces from the project source files and then build a minimal example around the main public types exposed by the package.
